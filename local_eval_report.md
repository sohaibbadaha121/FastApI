## Metric Explanations

| Metric | Method | What it measures |
| :--- | :--- | :--- |
| **Faithfulness** | Token overlap (answer ∩ context) | Is the answer grounded in retrieved passages? |
| **Context Recall** | ROUGE-L with Arabic pre-tokenization (ground_truth vs contexts) | Did the retriever find the right passages? |
| **Answer Relevancy** | Token overlap (question ∩ answer) | Does the answer address the question's key terms? |
| **Answer Correctness** | BERTScore F1 (answer vs ground_truth) | Is the answer semantically correct? |
